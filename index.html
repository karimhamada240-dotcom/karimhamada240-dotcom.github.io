HR Performance Analysis Dashboard

This project represents a complete business intelligence solution, transforming a raw HR dataset into an interactive and actionable dashboard. The workflow followed best practices, starting with data cleaning in Power Query, followed by the creation of a robust data model and custom metric calculations using DAX.

Key Responsibilities:

  • Data Preparation (Power Query): Performed essential data cleaning tasks, handled missing values, ensured consistent data types, and transformed columns with numerical codes into meaningful categories (e.g., Attrition Yes/No).
  • Data Modeling & Analysis (DAX): Developed critical measures to calculate key HR metrics, including Total Employees, Total Attrition, Attrition Rate (%), Average Years at Company, and Average Monthly Income.
  • Dashboard Design: Created a visually appealing and intuitive dashboard featuring KPI cards for an at-a-s-glance overview, multi-dimensional analysis charts, and interactive slicers for deep-dive analysis.

Tools Used:

Power BI
Power Query

Supermarket Sales Analysis Dashboard

This project demonstrates a complete business intelligence workflow, focused on analyzing supermarket sales data. The project followed a structured approach to transform raw data into a dynamic and insightful dashboard.

Key Responsibilities:

  • Data Preparation (Power Query): Imported the raw data and performed key cleaning steps, including removing irrelevant rows from the bottom of the table to ensure only valid data was used.
  • Data Modeling & Analysis (DAX): Created custom DAX measures to calculate Total Sales, Total Quantity, Average Retail Price, Average Order Quantity, and Average Tax.
  • Feature Engineering (DAX): Created a new column to calculate the number of days between the 'Order Date' and 'Ship Date', and a measure for the 'Average Shipment Days'.
  • Dashboard Design: Designed a dynamic dashboard to visualize key business metrics, including Total Sales by Customer, Total Sales by Month, and prominently displayed KPI cards.

Tools Used:

Power BI
Power Query
DAX

Global Sales Performance Analysis Dashboard

This project involved a comprehensive analysis of the 'Global Superstore' dataset. The primary objective was to build an interactive and visually compelling dashboard in Power BI to track sales, profitability, and geographical performance, transforming raw data into strategic business insights.

Key Responsibilities:

  • Data Cleaning & Transformation (Power Query): Corrected date format issues using the 'Change Type with Locale' feature to handle regional date differences. Verified all data types and ensured data integrity before loading.
  • Data Modeling: Implemented data modeling best practices by creating a dedicated Calendar table using DAX. Established and validated a correct one-to-many relationship between the Calendar table and the main sales data table.
  • Advanced DAX Measures: Developed a suite of DAX measures to calculate core KPIs, including Total Sales, Total Profit, and Profit Margin. Created advanced time-intelligence measures like `Last Year Sales` and `Sales YoY %` for comparative analysis.
  • Interactive Dashboard Design: Designed a professional, dark-themed dashboard featuring KPI cards with conditional formatting, a time-series line chart, a geographical map to visualize profit by country, and charts for categorical analysis. Implemented slicers for dynamic filtering by year, market, and category.

Tools Used:

Power BI
Power Query
DAX

Sales Data Analysis with DAX

This project showcases a complete business intelligence workflow, from data preparation to interactive visualization. The process involved cleaning raw data with Power Query and then building a comprehensive dashboard with KPIs and analytical reports using DAX.

Key Responsibilities:

  • Data Preparation (Power Query): Performed essential cleaning tasks to ensure data accuracy and created a new "Day of Purchase" column through feature engineering.
  • Data Modeling and Analysis (DAX): Developed custom measures to calculate critical business metrics, including Total Quantity, Max/Min Quantity Orders, Average Price per Unit, and Total Number of Orders.
  • Dashboard Creation: Designed a dynamic dashboard visualizing key insights, such as Total Quantity by Product Line, Total Orders by City/State, and Average Price by Product Line.

Power BI
